**2021年8月1号:**

#### 1.0.4更新小助手
- 配置文件对比1.0.2版本有变动,请添加以下到config.yml中或者删除旧config让重新生成
```
# 强化秘药 增加的强化概率 加算
secretMedicines: 4
# 强化秘药 可持续时间, 单位 秒
secretMedicinesTime: 1800
```
- 语言文件对比1.0.2版本有变动,zh_CN.yml中或者删除旧的让重新生成
```
# 强化秘药
secretMedicines:
  name: "&e强化秘药"
  lore:
    - ''
    - '&f[ &c使用效果 &f]'
    - ''
    - '&f>> 使用后,可以提升强化成功率'
    - '&f>> 效果可持续30分钟。'
```
