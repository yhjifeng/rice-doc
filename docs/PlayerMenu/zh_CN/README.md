## 一、简介
> PlayerMenu是一款有点好用的菜单插件
> 可以使用指令创建和编辑菜单-直接拖拽即可完成菜单配置

本插件由本人(MCBBS论坛ID:**ヽ米饭**)独立开发完成。

1. 支持**mysql**和本地**sqlite**存储数据
2. 支持变量
3. 支持点券和金币系统
4. 支持点击次数限制
5. 支持使用gui创建菜单
6. 支持使用gui编辑菜单
7. 支持自定义条件
8. 简单配置和方便更改
